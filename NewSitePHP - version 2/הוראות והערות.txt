הוראות:
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* בתיקיה הראשית נמצאים קבצי '.php' שהם הקבצים הראשיים של כל עמוד באתר (אלה קבצים שמפעילים את העמודים ולא נמצא בהם התוכן של כל עמוד).
   (!) הערה: בכל הקבצים הראשיים צריכות להיות אותן שורות קוד שמפעילות את התוכן הקבוע באתר, אך בכל קובץ תהיה בנוסף שורת קוד שמפעילה את העמוד שלו.

* תוכן כל דף נמצא בתיקיה 'PHP - HTML pages content' (בתוך קובץ '.php' שמשלב בתוכו קוד HTML).

* תוכן האתר הקבוע (כגון: כותרת, סרגל כלים, סרגל ימני, כותרת תחתונה) נמצא בתיקיה 'PHP - HTML static content' (בתוך קובץ '.php' שמשלב בתוכו קוד HTML).

* בתוך התיקיה 'PHP' נמצאים קטעי קוד בשפת PHP (שלא משלבים בתוכם HTML).
   (!) הערה: קטעי קוד אלה לא אחראיים על עיצוב האתר אלא אחראיים על ביצוע פעולות צד שרת (כגון: התחברות לבסיס נתונים, ביצוע התחברות/הרשמה לאתר).

* בתוך התיקיה 'JS' נמצאים קטעי קוד בשפת JavaScript שאחראיים על ביצוע פעולות צד לקוח (כגון: שינוי אלמנטים בדף, פתיחה/סגירה של חלונות).

* בתוך התיקיה 'CSS' נמצאים קבצים האחראיים על העיצוב של האתר.
   (!) הערה: הקבצים מחולקים לקובץ גלובלי, ולקבצים המתמקדים כל אחד בחלק מסויים של האתר (כותרת [header], גוף [main], תחתית [footer]).

* בתוך הקובץ '# HTML CODE TEMPLATE #' נמצאת תבנית של קוד ה - HTML של האתר במלואו עם סימון של כל חלק.

* בתוך התיקיה 'resources' נמצאים המשאבים של האתר (כגון: תמונות, וידאו, מוסיקה, מסמכים, הרחבות [כגון: Bootstrap]).




הערות כלליות:
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* כאשר יוצרים עמוד באתר צריך ליצור בתיקיה הראשית קובץ '.php' (שבו יהיו שורות קוד שמפעילות את העמוד [על פי ההוראות]),
   וצריך ליצור בנוסף קובץ '.php' בתיקיה 'PHP - HTML pages content' (שבו יהיה התוכן של העמוד בשפת HTML [וגם PHP אם רוצים לבצע התניה או בדיקה]).
   (!) הערה: צריך גם להוסיף כפתור לעמוד בסרגל הכלים העליון.

* הקובץ 'htmlHead.php' שבתיקיה 'PHP - HTML static content' הוא הקובץ שמכיל את תחילת קוד ה - HTML של האתר.

* הקובץ 'htmlBody_bottom.php' שבתיקיה 'PHP - HTML static content' הוא הקובץ שמכיל את סוף קוד ה - HTML של האתר.
   הוא יכיל את כל האלמנטים שלא קשורים למבנה הראשי של האתר (כגון: אלמנטים של אתחול סקריפט, אלמנטים של חלון קופץ).
